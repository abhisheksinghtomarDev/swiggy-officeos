# Roadmap

## Phase 1: MVP

Goal: prove that workplace food and pantry workflows can be handled safely through an AI copilot.

- Create workplace profile setup
- Build Slack command flow
- Add deployment-night ordering workflow
- Add pantry restock workflow
- Add admin approval before final actions
- Store basic workflow and approval history
- Validate Swiggy Food and Instamart MCP integration paths

## Phase 2: Pilot

Goal: make the experience useful for a small real team.

- Add Microsoft Teams support
- Add recurring pantry templates
- Add team preferences
- Add budget policy rules
- Add real-time workflow status dashboard
- Improve logs and monitoring

## Phase 3: Enterprise Readiness

Goal: support larger workplaces with stronger controls.

- Multi-office support
- Role-based admin controls
- Advanced approval policies
- Usage reporting
- Better tenant isolation checks
- More detailed audit exports
- Support and escalation flows
